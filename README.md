# MTC
This Is Ecommerce admin panel and node js API
##Setup

    1. add .env file in root 
        ALLOWED_ORIGINS=["*"]
        SWAGGER_HOST=
        SWAGGER_ENABLE=false
        CLIENT_ID=
        CLIENT_SECRET=
        SERVER_TIMEOUT=
        SERVER_PORT=
        ENCRYPT_IV=
        ENCRYPT_KEY=
        WEB=
        MONGO_URI=

    2. run sudo make deploy
